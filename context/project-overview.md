# Solution

SecureDoc AI automatically:

- extracts document text
- structures information
- compares historical records
- detects anomalies
- generates fraud risk reports
- provides underwriting insights

using:

- OCR
- Generative AI
- RAG
- anomaly detection
- intelligent document analysis

# Core Features

- PDF/Image upload
- OCR extraction
- Document comparison
- Financial anomaly detection
- Risk scoring
- AI-generated fraud explanation
- Real-time dashboard

# Technologies (JavaScript-Only Stack)

| Layer            | Technology                  | Purpose                         |
| ---------------- | --------------------------- | ------------------------------- |
| Frontend         | React 18+                   | UI & Dashboard                  |
| Backend          | Node.js + Express           | REST APIs & Logic               |
| **AI Engine**    | **LangChain.js**            | **LLM Orchestration & Agents**  |
| **OCR**          | **Tesseract.js**            | **Client-side Text Extraction** |
| **LLM Provider** | **Gemini API / OpenAI**     | **Generative AI**               |
| **RAG**          | **LangChain.js + ChromaDB** | **Vector Search & Context**     |
| Database         | MongoDB                     | Document Storage                |
| Vector DB        | ChromaDB (JS)               | Embeddings Storage              |
| Message Queue    | Bull/Redis                  | Async Processing                |
| **Status:**      | **Full Stack JS**           | **No Python Dependencies**      |
